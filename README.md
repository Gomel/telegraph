Telegraph Bot - Telegram text game bot

### To Development
`git clone https://github.com/devinterx/telegraph`

`cd telegraph`

`npm -g i webpack babel babel-cli babel-register`

`npm i`

### To Production
`git clone https://github.com/devinterx/telegraph`

`cd telegraph`

`npm i --production`

### ENV
`NTBA_FIX_319=true;BOT_TOKEN=<your-bot-token>`
